一.脚本介绍
    基于asyncio开发的高性能python异步扫描器,采用生产者-消费者的模式,支持自定义参数(详见config.json),只通过响应的状态码判断,需要自行准备字典文件
    python版本3.8+

二.config.json参数介绍
    url : 目标网址
    file : 字典路径(相对路径和绝对路径均可)
    random_UA : 随机user-agent池,自带,可自己修改
    suffixes : 后缀名,格式是.php .html这样的,最后需要一个空字符串(为了扫描目录本身)
    status : 白名单状态码
    header : 自定义请求头,如cookie之类的
    proxy : 自定义代理
    timeout : 超时控制
    max_worker : 消费者数,相当于线程数
    retries : 遇到网络波动等意外导致的丢包会重发,这是重发次数
    result_file : 保存结果的文件

三.使用方法
    1.安装依赖:
        pip install -r requirements.txt
    2.配置config.json
    3.运行main.py

四.优点
    1.高性能
        采用异步生产者-消费者的模式,提高扫描效率
    2.支持大字典,不爆内存
        生产者-消费者模式内存占用少,面对大文件也游刃有余
    3.代码结构清晰,易于配置

五.缺点
    1.只通过状态码来判断,可以自己在check函数里面增加判断
    2.不支持递归扫描
    3.无自带字典
    4.不支持多url
